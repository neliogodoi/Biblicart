Erro de Configuração do Firebase

Sua chave de API do Firebase não é válida. Parece que você ainda está usando os valores de exemplo.

Para corrigir isso, abra o arquivo src/firebase.config.ts no editor de código e substitua as credenciais de exemplo pelas credenciais reais do seu projeto Firebase.

// Em: src/firebase.config.ts

export const firebaseConfig = {{ '{' }}
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_STORAGE_BUCKET",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
{{ '}' }};

Você pode encontrar essas configurações no seu Console do Firebase em:
Configurações do Projeto (⚙️) > Geral > Seus apps > SDK setup and configuration.
